# 👞 ربات محاسبه بهای تمام‌شده کفش دست‌دوز

🌐 **فارسی** | [English](README.md)

ربات دوزبانه (فارسی/انگلیسی) پیام‌رسان **بله** برای محاسبه‌ی دقیق بهای تمام‌شده و قیمت فروش یک جفت کفش دست‌دوز.

🏛️ تعاونی کفاشان دست‌دوز تهران — تأسیس ۱۳۴۸

🔗 آدرس ربات: [@Shoe_cost_bot](https://ble.ir/Shoe_cost_bot)

---

## 📌 درباره ربات

این ربات به تولیدکنندگان کفش دست‌دوز کمک می‌کند هزینه‌ی واقعی تولید هر جفت کفش را گام‌به‌گام محاسبه کنند و حداقل قیمت فروش پیشنهادی را به‌دست آورند. تمام مبالغ برای **یک جفت کفش** وارد می‌شوند.

## ✨ امکانات

- انتخاب زبان فارسی یا انگلیسی در ابتدای کار
- پرسش گام‌به‌گام در سه بخش: مواد اولیه، دستمزد مستقیم، سایر هزینه‌ها
- انتخاب نوع جنس با دکمه (چرم، زیره، یراق‌آلات و…)
- دکمه‌ی «⬅️ قبلی» برای اصلاح وسط کار
- خلاصه‌ی قابل‌ویرایش پیش از محاسبه‌ی نهایی
- محاسبه‌ی خودکار قیمت تمام‌شده، سود و قیمت فروش
- پشتیبانی از ارقام فارسی

## 🧮 نحوه محاسبه

```
قیمت تمام‌شده = جمع مواد اولیه + جمع دستمزد + جمع سایر هزینه‌ها
سود          = قیمت تمام‌شده × (درصد سود ÷ ۱۰۰)
قیمت فروش    = قیمت تمام‌شده + سود
```

## 🚀 راه‌اندازی

1. از `@botfather` در بله یک ربات بسازید و توکن بگیرید.
2. کتابخانه‌ها را نصب کنید: `pip install -r requirements.txt`
3. توکن را در متغیر محیطی `BOT_TOKEN` قرار دهید.
4. اجرا: `python shoe_cost_bot.py`

## ☁️ استقرار روی Railway

1. مخزن را به [Railway](https://railway.app) متصل کنید (Deploy from GitHub repo).
2. در تب **Variables** متغیر `BOT_TOKEN` را با توکن بله بسازید.
3. Railway به‌صورت خودکار ربات را اجرا می‌کند.

> توجه: چون سرورهای بله داخل ایران هستند، در صورت قطعی اتصال، یک VPS ایرانی پایدارتر است.
